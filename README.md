# QQ Message Summary

A personal tool for collecting selected QQ group messages and summarizing unread/new messages with an AI API.

## Planned MVP

- Collect group messages from a local QQ adapter such as NapCatQQ.
- Store messages in SQLite.
- Track the last summarized message per group.
- Generate summaries with an OpenAI-compatible AI API.
- Provide a local web UI for choosing groups and reading summaries.
